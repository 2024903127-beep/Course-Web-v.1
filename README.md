# ArogyaX — Website

Healthcare consultation infrastructure platform website.

## Project Structure

```
arogyax-site/
├── index.html          ← Home page
├── solutions.html      ← Solutions (Hospitals, Clinics, Schools, Corporate, Public Health)
├── booths.html         ← Health Booth product page
├── integrations.html   ← API, SDK, Web & Mobile integration
├── security.html       ← Security architecture & compliance
├── contact.html        ← Contact form + demo request
├── css/
│   └── style.css       ← All styles (shared across pages)
├── js/
│   └── script.js       ← All JS (nav, form validation, animations)
└── assets/             ← Place any images here
```

## Deployment

This is a pure HTML/CSS/JS static site. Deploy to:

- **GitHub Pages**: Push the `arogyax-site/` folder contents to a `gh-pages` branch or the root of your repo
- **Netlify / Vercel**: Drag and drop the `arogyax-site/` folder
- **Any static host**: Just upload all files preserving the folder structure
